2025

Developed an end-to-end pipeline for processing and analyzing multimodal documents (PDFs, images, tables) using fine-tuned LLMs with Retrieval-Augmented Generation (RAG) capabilities.

The system reduced manual document processing time by 60% while maintaining 98% accuracy across various document types. Implemented PEFT (Parameter-Efficient Fine-Tuning) techniques to reduce model size by 40% without sacrificing performance.

The pipeline was deployed on AWS SageMaker with autoscaling, handling up to 10,000 documents per day with average latency under 500ms.

Telecom Churn Dashboard

2023

Developed a predictive analytics dashboard that identified at-risk telecom customers with 85% accuracy, enabling targeted retention campaigns that reduced churn by 18%.

The system processed real-time customer data streams to update churn risk scores hourly. Implemented SHAP value explanations to help customer service representatives understand risk factors.

The dashboard was built with Streamlit and deployed on AWS, serving over 500 customer service agents with sub-second latency. Integrated with the company's CRM to automatically trigger retention workflows for high-risk customers.
